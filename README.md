# Personal Task Manager - Task Master
# 🗂️ Task Manager App

A full-stack Task Management web application built using **React (Frontend)** and **Node.js + Express (Backend)**.

---

## 🚀 Features

* ➕ Add new tasks
* 📝 Edit tasks
* ❌ Delete tasks
* 📌 Mark tasks as complete/incomplete
* 🔄 Real-time updates between frontend & backend
* 📱 Responsive UI

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Vite
* CSS

**Backend:**

* Node.js
* Express.js
* JSON file (as database)

---

## ⚙️ Installation & Setup

### 1. Clone repository

```
git clone https://github.com/yashgupta-101/Task-Manager.git
```

### 2. Install dependencies

#### Frontend

```
cd client
npm install
npm run dev
```

#### Backend

```
cd server
npm install
node server.js
```
